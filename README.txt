# Terry's Auto Mechanics LLC Website

This package includes:

- `index.html` — main company homepage
- `authorization.html` — customer authorization and consent form
- `style.css` — black/red professional styling
- `app.js` — signature pad, local save, print/save PDF, email summary button

## How to upload to GitHub Pages

1. Open your website repository:
   `terryvaughn.github.io/terrysautomechanics.github.io`

2. Upload these files into the main/root folder:
   - index.html
   - authorization.html
   - style.css
   - app.js

3. Commit/save the changes.

4. Your authorization page should be available at:
   `https://terryvaughn.github.io/terrysautomechanics.github.io/authorization.html`

## Important note

GitHub Pages is static hosting. That means it cannot automatically store forms on a server or email PDFs by itself.

This version works immediately by letting the customer:
1. Fill out the authorization form
2. Sign it
3. Press Submit
4. Press Print / Save as PDF
5. Press Email Summary

For automatic form storage and automatic emails, connect the form to a service like Formspree, Jotform, Google Forms, Netlify Forms, or a custom backend.
